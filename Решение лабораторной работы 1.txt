 Класс Product (Продукт)
Это основа программы - "шаблон" для создания продуктов.
Приватные поля (к ним нельзя обращаться напрямую извне):
cppprivate:
    string name;              // Название продукта
    string productionDate;    // Дата производства
    string manufacturer;      // Производитель
    double weight;            // Вес в килограммах
Почему приватные? Это инкапсуляция - один из принципов ООП. Мы защищаем данные от случайного изменения.
Публичные методы (функции класса):

Конструктор без параметров Product() - создаёт "пустой" продукт
Конструктор с параметрами Product(...) - создаёт продукт сразу с данными
Деструктор ~Product() - вызывается автоматически при удалении объекта
Геттеры (getName(), getWeight() и т.д.) - получают значения приватных полей
Сеттеры (setName(), setWeight() и т.д.) - устанавливают новые значения
input() - вводит данные с клавиатуры
display() - красиво выводит продукт на экран
hasManufacturer() - проверяет, указан ли производитель


 Глобальная переменная
cppvector<Product> products;
Это наш массив продуктов. vector - это умный динамический массив из STL (стандартной библиотеки C++). Он автоматически меняет размер при добавлении/удалении элементов.

 Функции меню (подробно)
1. Ввод элементов массива (inputProducts())
Что делает:

Спрашивает, сколько продуктов хочешь ввести
Очищает старый массив
В цикле вводит каждый продукт
Добавляет их в вектор

Пример использования:
Введите количество продуктов: 2

--- Продукт 1 ---
Введите название: Молоко
Введите дату производства: 20.01.2026
Введите производителя: Простоквашино
Введите вес (кг): 1.5

2. Вывод элементов массива (displayProducts())
Что делает:

Проверяет, не пуст ли массив
Красиво форматирует вывод в виде таблицы
Использует setw() для выравнивания колонок

Вывод:
=== СПИСОК ПРОДУКТОВ ===
Название            Дата           Производитель       Вес       
-----------------------------------------------------------------
1. Молоко            20.01.2026     Простоквашино       1.5 кг
2. Хлеб              19.01.2026     Каравай             0.5 кг

3. Сортировка по названию (sortByName())
Что делает:

Использует функцию sort() из библиотеки <algorithm>
Сравнивает названия с помощью лямбда-функции:

cpp  [](const Product& a, const Product& b) {
      return a.getName() < b.getName();
  }
Это короткая анонимная функция, которая говорит: "сортируй по возрастанию названий"
Результат: Продукты упорядочены от А до Я (а до я в русском алфавите).

4. Вывод продуктов с производителем (displayWithManufacturer())
Что делает:

Проходит по всем продуктам
Показывает только те, у которых поле "производитель" не пустое
Если ничего не найдено - сообщает об этом

Логика:
cppif (p.hasManufacturer()) {  // Если производитель указан
    p.display();             // Показать продукт
}

5. Добавить элемент перед заданным весом (addBeforeWeight())
Что делает:

Спрашивает вес продукта, перед которым нужно вставить новый
Ищет этот продукт в массиве
Вставляет новый элемент перед найденным
Использует insert() - метод вектора

Пример:
Есть: [Молоко(1.5кг), Хлеб(0.5кг), Масло(0.2кг)]
Вводим вес: 0.5
Добавляем: Сыр(0.3кг)
Результат: [Молоко(1.5кг), Сыр(0.3кг), Хлеб(0.5кг), Масло(0.2кг)]
                              ↑ вставлено сюда

6. Изменить последний элемент (modifyLastProduct())
Что делает:

Берёт последний элемент массива через products.back()
Показывает его текущие данные
Позволяет ввести новые данные

Это полезно для: Исправления ошибок в последнем введённом продукте.

7. Записать в файл (saveToFile())
Что делает:

Создаёт файл products.txt (или перезаписывает, если он есть)
Сначала записывает количество продуктов
Затем для каждого продукта записывает все поля по строкам

Формат файла:
3
Молоко
20.01.2026
Простоквашино
1.5
Хлеб
19.01.2026
Каравай
0.5
...
Зачем это нужно? Чтобы данные сохранились после закрытия программы!

8. Вывести содержимое файла (displayFromFile())
Что делает:

Открывает файл products.txt
Читает количество продуктов
Считывает данные каждого продукта
Выводит их в виде таблицы

Важно: Эта функция не изменяет массив в памяти, она только читает из файла и показывает.

 Главная функция main()
Основной цикл программы:
cppdo {
    showMenu();          // Показать меню
    cin >> choice;       // Получить выбор
    
    switch (choice) {    // Выполнить функцию
        case 1: inputProducts(); break;
        case 2: displayProducts(); break;
        // и так далее...
    }
    
} while (choice != 0);   // Пока не выбран выход
Это меню с повторением - программа работает, пока пользователь не выберет "0" (выход).

 Ключевые концепции ООП, которые здесь использованы
1. Инкапсуляция
Приватные поля защищены, доступ только через геттеры/сеттеры.
2. Конструкторы
Создают объект в нужном состоянии (пустой или с данными).
3. Методы класса
Функции, которые работают с данными объекта (input(), display()).

Как компилировать и запускать
Компиляция:
bashg++ -o products program.cpp
Запуск:
bash./products        # Linux/Mac
products.exe      # Windows
